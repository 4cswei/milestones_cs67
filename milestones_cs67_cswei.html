Instructions

  1. Create a new Codepen that has at least the following:
    • A head with proper meta (viewport, title, description, etc)
    • A header
    • A paragraph
    • A form (with all fields we learnt in class and two we didn’t learn using docs)
    • An unordered list and ordered list (nested if you wish)
    • An image with a link wrapped around it
    • A table using elements: table, thead, tbody, tr, th, td
  2. Feel free to create as much as you want on top of that

The more you practice, the easier it is going to be to proceed with more advanced classes!

Don’t be afraid to explore new elements using docs and experiment with different attributes.
